# PowerShell 2401 实用工具箱

这是一个基于 PowerShell 的多功能脚本库，旨在简化视频处理、文件管理和 AI 任务。通过统一的入口命令，你可以轻松调用各种强大的后端工具（如 FFmpeg、Whisper 等）。

## 🚀 快速开始

### 1. 安装与配置
首先，你需要将项目目录添加到系统的环境变量中。

1. 以管理员权限打开 PowerShell。
2. 进入项目根目录。
3. 执行安装脚本：
   ```powershell
   pwsh ./pw2401.ps1 install
   ```
4. **重启终端**即可开始使用 `pw2401` 命令。

### 2. 基本用法
所有命令都遵循以下格式：
```powershell
pw2401 <命令名称> [参数]
```
如果不带任何参数执行 `pw2401`，它将显示帮助信息。

## 🛠️ 核心功能模块

### 🎬 视频与图像处理 (FFmpeg)
这些命令封装了复杂的 FFmpeg 参数，让你通过简单的命令完成高质量任务。

*   **`ff-masonry` (瀑布流拼接)**：将多张图片拼接成精美的瀑布流长图。
    *   *示例*：`pw2401 ff-masonry -ColumnCount 3`
*   **`ff-svt` / `ff-nv-av1` (高效编码)**：使用 CPU (SVT-AV1) 或 NVIDIA 显卡 (NVENC) 进行高性能 AV1 视频压缩。
    *   *示例*：`pw2401 ff-svt -Preset 5 -CRF 32`
*   **`ff-vmaf` (视频质量分析)**：计算转码后视频与原片的相似度评分。
    *   *示例*：`pw2401 ff-vmaf 源目录 目标目录`
*   **`ff-info`**：快速获取视频流信息。

### 📂 目录与文件管理
高效处理海量文件，保持电脑整洁。

*   **`dir-info`**：深度分析目录占用空间，支持多种视图。
*   **`dir-trim` / `dir-void`**：自动清理空文件夹或冗余的系统生成文件。
*   **`dir-zip`**：自动扫描目录并根据规则进行压缩备份。
*   **`hard-link`**：批量创建硬链接，在不增加磁盘占用的情况下整理文件。

### 🎙️ AI 语音识别 (Whisper)
集成了主流的快速 Whisper 实现。

*   **`whisper-faster` / `whisper-ctranslate2`**：将音视频文件高精度转换为字幕或文本。

## 📖 进阶提示

*   **查看帮助**：如果你不确定某个命令怎么用，可以直接在命令后加 `--help` (部分脚本支持) 或直接阅读 `functions/` 目录下的 `.help.txt` 文件。
*   **自定义配置**：项目根目录下的 `config.json` 可以配置默认的后端工具路径（如 FFmpeg 的位置）。
*   **调试日志**：如果遇到问题，可以查看生成的日志文件以排查错误。
